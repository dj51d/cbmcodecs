# Description:
#
#   This file lists the CUS assignments used for encoding PETSCII
#   characters in Unicode. The assignments start at U+F100 in order
#   to avoid conflicts with the already populated subareas:
#
#   U+F000 -- U+F0FF  used by Microsoft Corporation symbol font
#   U+F600 -- U+F7FF  used by Adobe Corporation
#   U+F800 -- U+F8FF  used by Apple Computer Inc
#
#   Unicode and glyph name is separated by a semicolon ";" optionally
#   followed by an alternate Unicode.
#
#   Lines starting with "#" are comments; blank lines should be ignored.
#
# Revision History:
#
#   1.0  [26 Nov 2000]
#        Initial work on a PETSCII CUS Area started. Assigned the first
#        characters to represent C64 PETSCII codes. Other Commodore
#        models codes to be assigned in higher areas.
#
#   1.1  [10 May 2002]
#        Added MEDIUM SHADE SLASHED LEFT and MEDIUM SHADE SLASHED RIGHT
#        the former overriding the old entry MEDIUM SHADE SLASHED
